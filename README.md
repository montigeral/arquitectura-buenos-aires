# Buenos Aires Arquitectónico 🏛️

## Descripción del Proyecto
Este repositorio contiene el código fuente del sitio web "Buenos Aires Arquitectónico", desarrollado como proyecto integrador para la Fase 5 del curso de Programación Web de la UNAD. 

## Finalidad del Sitio
El objetivo principal del sitio es promover el turismo arquitectónico, ofreciendo una experiencia interactiva y accesible. El proyecto incluye la internacionalización del contenido (disponible en español e inglés) y fue diseñado aplicando buenas prácticas de maquetación, responsividad y control de versiones.

## Tecnologías Utilizadas 🛠️
* **Estructura:** HTML5
* **Estilos:** CSS3
* **Interactividad:** JavaScript
* **Control de Versiones:** Git y GitHub
* **Despliegue:** GitHub Pages
